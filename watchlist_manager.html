📤 Export — watchlist.json

Copy JSON ด้านล่างแล้วนำไปแทนไฟล์ watchlist.json ใน GitHub repository

📥 Import — วาง JSON ที่นี่

วาง JSON จากไฟล์ watchlist.json ที่มีอยู่เพื่อโหลด Watchlist เดิม

⚙️ วิธีใช้งาน
1. เพิ่มผู้เล่นที่ tab "เพิ่มผู้เล่น"
2. Export JSON แล้ว commit ไฟล์ watchlist.json ลง GitHub repo
3. GitHub Actions จะรันทุก 30 นาที และเช็คแต่ละ match ใหม่
4. เมื่อเจอผู้เล่นใน Watchlist → แจ้งเตือนผ่าน Line ทันที 🎉

Secrets ที่ต้องตั้งใน GitHub:
MY_DOTA_ID = Steam32 Account ID ของคุณ
LINE_TOKEN = Line Notify Token
